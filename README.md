# MujinOS Documentation System PoC

This repository serves as a professional demonstration for the **Lead Documentation Engineer** role at Mujin. It showcases a modern **Docs-as-Code** ecosystem designed to bridge the gap between industrial automation and software development.

## 🚀 Key Features
* **Persona-Driven Architecture:** Dedicated navigation paths for Industrial Operators (Safety/Hardware) and Software Developers (APIs).
* **Single Source of Truth:** Automated API reference generation using `mkdocstrings`, pulling directly from the Python source code in `/src`.
* **CI/CD Pipeline:** Automated build and deployment to GitHub Pages via GitHub Actions on every push.
* **Industrial Standards:** Integrated ISO-compliant safety admonitions and technical formatting.

## 🛠 Tech Stack
* **SSG:** MkDocs Material
* **Automation:** GitHub Actions
* **API Engine:** Python (mkdocstrings)
* **Linting/Quality:** Vale (Prose linting)
