# Documentation Style Guide

This guide ensures consistency across all MujinOS documentation.

## Product Terminology
| Term | Do Not Use | Context |
| :--- | :--- | :--- |
| **MujinOS** | Mujin OS, mujin os | Our primary operating system. |
| **Robotic Arm** | Robot, Bot | Refer to the physical hardware precisely. |
| **Teaching Pendant** | Remote, Controller | The handheld HMI device. |

## Voice and Tone
* **Direct and Active:** Use "Press the button" instead of "The button should be pressed."
* **Safety First:** Safety instructions must precede the step they describe.
